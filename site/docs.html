Документация проекта

Подробная информация о проекте и его реализации

Проблематика проекта

Студенты сталкиваются с трудностями при поиске и обработке информации при работе с документами, что негативно влияет на образовательный процесс. Это затрудняет усвоение знаний и подготовку будущих специалистов к реальным условиям работы. Разработка интеллектуальных информационно-измерительных систем для автоматизации поиска и актуализации документов может значительно улучшить качество образования и подготовить студентов к решению практических задач.

Актуальность проекта

Интеллектуальные информационно-измерительные системы - активно развивающееся направление в сфере метрологии и ИТ, которые в свою очередь требуют актуализацию нормативно-технических и правовых документов. Отсутствие такой базы негативно влияет на качество образовательных процессов для будущих специалистов.

Цель проекта

Разработка прототипа десктоп-приложения с системы нормативно-технической и правовой базы совместно с решениями АО "Кодекс" для обеспечения высокого качества образовательных процессов.

Ключевые задачи проекта

  1. Исследование учебного плана и создание нормативно-правовой базы данных.
  2. Создание дизайна для прототипа десктоп-приложения.
  3. Разработка прототипа десктоп-приложения.
  4. Тестирование и отладка готового прототипа десктоп-приложения.

Этапы реализации проекта

  1. Составление списка дисциплин по учебному плану.
  2. Анализ дисциплин и составление списка скачиваемых документов.
  3. Выгрузка нормативных документов из системы «Техэксперт».
  4. Разработка прототипа десктоп-приложения.
  5. Проведение пользовательского тестирования дизайна.
  6. Финализация дизайна с учетом полученных отзывов.
  7. Составление технического задания на разработку прототипа.
  8. Выбор технологий и инструментов для разработки.
  9. Интеграция дизайна и пользовательского интерфейса.
  10. Реализация функционала приложения согласно техническому заданию.
  11. Проведение функционального и юзабилити-тестирования.
  12. Исправление выявленных ошибок и доработка функционала.

Описание полученных результатов

Период выполнения: 26.03.2025 - 12.05.2025

Командой проекта «Интеграция решений АО «Кодекс» с нормативно-технической базой кафедры» были достигнуты следующие результаты:

  1. Составлен список дисциплин по учебному плану направления 27.03.02 Управление качеством
  2. Проанализированы дисциплины, составлены списки скачиваемых документов и их выгрузка из системы «Техэксперта»
  3. Разработан макет десктоп-приложения с основными функциями
  4. Составлены, выложены и проанализированы опросники для преподавателей и обучающихся
  5. Интегрирован дизайн и пользовательский интерфейс в десктоп-приложение
  6. Реализован функционал приложения согласно техническому заданию

Промежуточный продуктовый результат

Промежуточный продуктовый результат включает в себя прототип десктоп-приложения «Интеграция решений АО «Кодекс» с нормативно-технической базой кафедры» с реализованными базовыми функциями.
